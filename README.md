# Face

A small **Three.js** experiment that renders an interactive 3D face.  

---

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/Satko147/Face.git
cd Face
npm install
```

Run it via:

```bash
npm run dev
```

Project should run on: http://localhost:1234
